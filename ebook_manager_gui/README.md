# 电子书管理器

一个跨平台的 GUI 电子书管理器，支持 Windows、Linux 和 Mac 系统。

## 功能特性

- 📚 **书架模式**: 以表格列表形式展示所有电子书
- 🔍 **搜索功能**: 支持对标题、副标题、作者、分类、物理位置、扩展名、ISBN 进行模糊查询
- 🔄 **排序功能**: 点击列头可按该列升序/降序排序
- 📥 **导入功能**: 
  - 拖拽文件或文件夹到导入窗口
  - 点击按钮选择文件或文件夹
  - 递归扫描目录，自动跳过已存在的文件
- 📖 **支持格式**: EPUB、MOBI、AZW3
- ✏️ **详情管理**:
  - 单击列表项打开详情窗口
  - 支持修改所有字段信息
  - 支持删除书籍记录（不会删除源文件）
- 🚀 **快速打开**: 双击列表项用系统默认应用打开电子书
- 🖼️ **封面显示**: 自动提取并显示 EPUB 封面

## 项目结构

```
ebook_manager_gui/
├── main.py              # 程序入口
├── main_window.py       # 主窗口
├── import_window.py     # 导入窗口
├── detail_window.py     # 详情窗口
├── database.py          # 数据库操作
├── ebook_parser.py      # 电子书解析器
├── requirements.txt     # 依赖包列表
├── covers/              # 封面图片存储目录（自动创建）
└── ebook_manager.db     # SQLite 数据库文件（自动创建）
```

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

## 使用说明

### 导入电子书

1. 点击菜单 `文件` -> `导入电子书`
2. 在导入窗口中：
   - 将文件或文件夹拖拽到虚线区域
   - 或点击 "选择文件" / "选择文件夹" 按钮
3. 点击 "开始导入" 按钮

### 查看和编辑书籍信息

1. **单击** 列表中的任意位置打开详情窗口
2. 在详情窗口中：
   - 点击 "修改" 按钮进入编辑模式
   - 修改完成后点击 "保存"
   - 点击 "删除" 按钮删除该书籍记录

### 打开电子书

- **双击** 列表中的任意位置，将使用系统默认应用打开该电子书

### 搜索书籍

- 在顶部搜索框输入关键词，将实时过滤匹配的书籍

### 排序

- 点击任意列标题，将按该列排序。再次点击切换升序/降序。

## 数据库说明

使用 SQLite 数据库，存储以下字段：

- `id`: 主键
- `title`: 标题
- `subtitle`: 副标题
- `author`: 作者
- `category`: 分类
- `file_size`: 文件大小（字节）
- `physical_path`: 物理路径
- `extension`: 扩展名
- `isbn`: ISBN
- `rating`: 评分（0-5）
- `douban_url`: 豆瓣地址
- `notes`: 备注
- `cover_path`: 封面图片路径
- `created_at`: 创建时间
- `updated_at`: 更新时间

## 技术栈

- **GUI 框架**: PyQt6
- **数据库**: SQLite3
- **EPUB 解析**: ebooklib
- **图像处理**: Pillow
- **HTML 解析**: BeautifulSoup4

## 注意事项

1. 删除书籍记录只会删除数据库中的信息，不会删除源文件
2. 封面图片会自动保存到 `covers` 目录
3. 数据库文件 `ebook_manager.db` 会在程序首次运行时自动创建
4. MOBI 和 AZW3 格式的元数据解析支持有限，主要依赖文件名
