# 电子书管理器 v2.0

基于 PyQt6 的跨平台电子书管理工具。

## 功能特性

- 📚 书架展示：表格列表展示所有电子书，含封面缩略图
- 🔍 搜索功能：支持标题、副标题、作者、ISBN、分类、标签等多字段模糊查询
- 🔄 排序功能：点击列头按该列升序/降序排序
- 📥 导入功能：支持拖拽、文件选择、文件夹选择，自动跳过已存在的文件
- 📖 格式支持：EPUB、PDF
- ✏️ 详情管理：查看和修改所有字段信息，支持删除书籍记录
- 🚀 快速打开：双击用系统默认应用打开电子书
- 🖼️ 封面显示：自动提取 EPUB 和 PDF 封面
- 🌐 豆瓣解析：从豆瓣获取书籍详细信息（使用 lxml + XPath 稳定解析）
- ✅ 批量操作：支持批量选择、批量解析豆瓣信息、批量删除

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 配置豆瓣 Cookie

1. 在浏览器中登录 [豆瓣](https://www.douban.com)
2. 按 F12 打开开发者工具，切换到 Network（网络）标签
3. 刷新页面，点击任意请求
4. 在请求头中复制完整的 Cookie 值
5. 在程序中点击 `设置` → `豆瓣配置`，粘贴 Cookie 并保存

## 支持的书籍字段

| 字段 | 说明 |
|------|------|
| title | 标题 |
| subtitle | 副标题 |
| authors | 作者（逗号分隔） |
| publisher | 出版社 |
| pubdate | 出版日期 |
| isbn | ISBN 编号 |

| tags | 标签（逗号分隔） |
| series | 丛书 |

| douban_url | 豆瓣链接 |
| douban_id | 豆瓣 ID |

| notes | 个人备注 |

| cover_url | 封面网络地址 |
| file_size | 文件大小 |
| physical_path | 文件路径 |
| extension | 扩展名 |
| page_count | 页数 |
| parse_status | 解析状态 |

## v2.0 更新说明

### 相比 v1.0 的改进：

1. **移除了 MOBI/AZW3 格式支持**
   - 这两种格式的元数据解析支持有限，封面提取困难

2. **新增 PDF 格式支持**
   - 提取 PDF 元数据（标题、作者、出版社、主题等）
   - 自动提取 PDF 首页图片作为封面
   - 显示 PDF 页数信息

3. **重新实现豆瓣信息解析**
   - 采用 fetch_info 项目的实现方式
   - 使用 lxml + XPath 替代正则表达式，更稳定
   - 使用豆瓣通用搜索页面，比 API 更可靠
   - 增加随机延迟和完善的请求头设置
   - 支持缓存最近查询结果，减少重复请求

4. **扩充数据库字段**
   - 新增：副标题、出版社、出版日期、豆瓣ID、封面URL
   - 新增：丛书、页数、语言、标签、解析状态、最后解析时间
   - 作者字段改为支持多人（逗号分隔）
   - 标签字段支持多标签（逗号分隔）

5. **优化用户界面**
   - 新增出版社、出版日期、标签列
   - 详情窗口增加出版社、出版日期、标签、丛书、页数等字段
   - 批量操作按钮更醒目
   - 豆瓣解析状态实时显示

## 注意事项

- 删除书籍记录只会删除数据库中的信息，不会删除源文件
- 封面图片会自动保存到 `covers` 目录
- 数据库文件 `ebook_manager.db` 会在程序首次运行时自动创建
- 豆瓣解析需要配置有效的 Cookie，且有访问频率限制
- 建议定期备份数据库文件和封面目录
