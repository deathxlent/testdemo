# PONG - 1972 Arcade Edition (LAN Multiplayer)

严格遵循1972年街机版PONG游戏规则的本地局域网联机游戏。

## 游戏特性

- ✅ 1972年街机版完整规则
- ✅ 板不同部位不同回弹角度
- ✅ 时间限制（100秒）
- ✅ 获胜条件（先得11分）
- ✅ 原版音效
- ✅ 本地局域网联机
- ✅ 玩家在线列表显示
- ✅ 胜负平统计
- ✅ 双击IP开始对战

## 安装运行

### 前置要求
- Node.js (v14 或更高版本)

### 安装步骤

1. 安装依赖：
```bash
npm install
```

2. 启动服务器：
```bash
npm start
```

3. 打开浏览器访问：
- 本机: `http://localhost:3000`
- 局域网其他电脑: `http://[你的IP地址]:3000`

## 游戏规则（1972年街机版）

1. **胜负条件**: 先得到11分的玩家获胜
2. **时间限制**: 每局游戏100秒，时间结束时分数高的获胜
3. **回弹角度**: 
   - 球拍中心击球：直线回弹
   - 球拍边缘击球：大角度回弹
   - 球速每次击球增加3%
4. **控制**: W/S键 或 上下方向键移动球拍

## 操作说明

1. 游戏启动后自动加入玩家列表
2. 玩家随机分配到左/右侧
3. 双击对方玩家IP开始新比赛
4. 玩家列表显示W(胜)/L(负)/D(平)统计
